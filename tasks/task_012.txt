# Task ID: 12
# Title: Implement Content Sharing Functionality
# Status: done
# Dependencies: 3, 7
# Priority: medium
# Description: Develop the ability to share existing platform content (courses, blog posts, products) to the feed with optional commentary.
# Details:
Implement the following features:

1. Share Dialog:
   - Create modal/popup for sharing content
   - Allow adding commentary text
   - Provide visibility options
   - Show preview of content being shared

2. Content Types Support:
   - Blog posts: Show title, excerpt, featured image
   - Courses: Show title, description, cover image
   - Products: Show name, image, price
   - Events: Show title, date, location
   - Group posts: Respect original visibility settings

3. Share Button Integration:
   - Add share buttons to relevant content across the platform
   - Implement consistent share flow

4. Shared Content Display:
   - Design card layout for shared content
   - Show original content preview with link
   - Display sharer's commentary
   - Indicate content type and source

5. Analytics:
   - Track share counts for content
   - Record share sources and engagement

Ensure proper permission checking to prevent sharing of restricted content. Implement OpenGraph-style previews for shared content.

# Test Strategy:
1. Test sharing different content types
2. Verify commentary is saved correctly
3. Test visibility settings work
4. Verify shared content displays properly in feed
5. Test permission restrictions
6. Verify analytics tracking
7. Test share functionality across different platform sections
